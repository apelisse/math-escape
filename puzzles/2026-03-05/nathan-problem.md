# Nathan's Daily TV Code Challenge

## Concept of the Day
Exponents are evaluated before multiplication or division. When bases are the same, use exponent rules; when bases differ, rewrite them to a common base to simplify.

## Instructions
Solve each expression for A, B, C, and D. Use PEMDAS and work inside parentheses first. Each answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $\frac{4^5}{2^8}$
- B = $\frac{3^6}{3^4} - \frac{2^5}{2^4}$
- C = $\frac{6^4}{2^3 \times 3^3}$
- D = $\frac{2^{10}}{4^3 \times 2^1}$
