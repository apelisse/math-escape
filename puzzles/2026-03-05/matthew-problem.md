# Matthew's Daily TV Code Challenge

## Concept of the Day
A factorial means you multiply a whole number by all the smaller whole numbers down to 1. For example, $5! = 5 \times 4 \times 3 \times 2 \times 1$. Factorials often cancel when they appear in fractions.

## Instructions
Solve each expression for A, B, C, and D. Remember PEMDAS: do factorials before multiplication or division. Each answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $\frac{6!}{4! \times 5}$
- B = $\frac{5!}{3! \times 2!} - 1$
- C = $\frac{6!}{5!} + \frac{4!}{3! \times 4}$
- D = $\frac{8!}{6! \times 7}$
