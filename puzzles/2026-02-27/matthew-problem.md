# Matthew's Daily TV Code Challenge

## Concept of the Day
When you multiply powers with the same base, add the exponents. When you divide, subtract the exponents. For a power of a power, multiply the exponents: $(a^m)^n = a^{m \times n}$.

## Instructions
Solve each expression for A, B, C, and D. Remember PEMDAS: exponents before multiplication or division. Each answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $(2^5 \times 2^2) \div (2^3 \times 2^1)$
- B = $(3^4 \times 3^1) \div 3^3$
- C = $(5^3 \div 5^1) \div 5^1$
- D = $(4^2)^2 \div 4^3$
