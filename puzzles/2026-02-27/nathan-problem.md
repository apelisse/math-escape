# Nathan's Daily TV Code Challenge

## Concept of the Day
Do additions inside parentheses first, then multiply or divide. You can simplify by canceling common factors in fractions.

## Instructions
Solve each expression for A, B, C, and D. Use PEMDAS: parentheses and additions before multiplication or division. Each answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $\frac{2 \times 3 \times 5}{3 \times 5} + \frac{(3 \times 5) \times 4}{2 \times 3 \times 5}$
- B = $\frac{1 + 1 \times 3}{3 \times 8} \times \frac{(3 \times 2) \times (5 + 3)}{4 \times 2}$
- C = $\frac{(2 + 1) \times 4}{3 \times 4} + \frac{(3 \times 4) \times 2}{(2 + 1) \times 4}$
- D = $\frac{(5 + 1) \times 3}{2 \times 3} + \frac{(2 \times 3) \times 3}{(5 + 1) \times 3}$
