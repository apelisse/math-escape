# Matthew's Daily TV Code Challenge

## Concept of the Day
A factorial means you multiply a whole number by all the smaller whole numbers down to 1. For example, $5! = 5 \times 4 \times 3 \times 2 \times 1$. This makes big products easier to write, and sometimes parts cancel right away in a fraction.

## Instructions
Solve each expression for A, B, C, and D. Remember PEMDAS: do factorials before multiplication or division. Each answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $3! + 2!$
- B = $4! \div 3!$
- C = $5! \div (3! \times 4)$
- D = $9! \div 8!$
