# Nathan's Daily TV Code Challenge

## Concept of the Day
Exponents show repeated multiplication. For example, $2^3$ means $2 \times 2 \times 2$. When you multiply powers with the same base, add the exponents: $2^a \times 2^b = 2^{a + b}$. When you divide powers with the same base, subtract the exponents: $\frac{2^a}{2^b} = 2^{a - b}$.

## Instructions
Solve each expression for A, B, C, and D. Remember PEMDAS: exponents before multiplication or division. Each answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $2^3 \div 2^1$
- B = $(1 + 2)^2 \div 3^1$
- C = $(4^2 \times 4^1) \div 4^2$
- D = $(5^{20} \times 5^3) \div 5^{22}$
