# Matthew's Daily TV Code Challenge

## Concept of the Day
When the same base appears on the top and bottom of a fraction, you can
cancel by subtracting exponents. You still add exponents when powers
with the same base are multiplied, and for a power of a power you
multiply the exponents: $(a^m)^n = a^{m \times n}$. Sometimes it also
helps to rewrite a power with a different base, like $4^2 = (2^2)^2 = 2^4$.

## Instructions
Solve each expression for A, B, C, and D. Remember PEMDAS: exponents
before multiplication or division. Look for powers with the same base on
the top and bottom so you can cancel before doing the final power, and
if you divide by a fraction, flip the second fraction and multiply. Each
answer is a single digit (0-9). The TV code is ABCD.

## Expressions
- A = $\frac{2^4 \times 3^5 \times 5^2}{2^1 \times 3^5 \times 5^2}$
- B = $\frac{(3^2)^3 \times 2^4}{3^5 \times 2^4}$
- C = $\frac{3^100}{9^49}$
- D = $\frac{2^12 \times 3}{2^2 \times 5} \div \frac{2^9 \times 6}{3 \times 10}$
